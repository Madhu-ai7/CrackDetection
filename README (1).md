# Crack Detection System

## Project Overview
This project is a computer vision-based crack detection system developed using deep learning. The system classifies surface images as either cracked or non-cracked. It can be used for building inspection, road surface monitoring, and structural maintenance support.

## Aim
The aim of this project is to develop an AI-based system that can automatically detect cracks from image data.

## Technologies Used
- Python
- Google Colab
- TensorFlow / Keras
- NumPy
- Matplotlib
- OpenCV

## Dataset
The dataset contains images of cracked and non-cracked surfaces. Images were resized and preprocessed before training the model.

## Model
A convolutional neural network / transfer learning model was trained to classify images into:
- Crack
- No Crack

## System Workflow
1. Upload surface image
2. Preprocess image
3. Load trained model
4. Predict class
5. Display result with confidence score

## How to Run
1. Open the notebook in Google Colab.
2. Mount Google Drive if the model is stored there.
3. Upload a test image.
4. Run the prediction cell.
5. View the predicted result and confidence score.

## Results
The system successfully detects cracks from sample images and displays prediction confidence.
